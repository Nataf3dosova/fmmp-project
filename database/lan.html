Лань

Контент базы знаний для технических вузов

На сайте Лань пользователям доступны два раздела — Книги и Журналы. В глобальных разделах содержатся категории, в которых, в свою очередь, расположены подкатегории. Категории, в которых есть доступные для чтения документы, выделены в меню синим цветом. Если в разделе нет доступных для чтения документов, раздел серый.

Контент коллекции для СПО

Если вам необходимы учебные материалы только для среднего профессионального образования, выберите опцию «Каталог СПО» над меню каталога книг. После этого в меню будут показаны учебные материалы только для СПО.

Доступ и регистрация

Доступ может осуществляться как из сети БНТУ, так и вне ее — через личный кабинет.

Для получения удаленного доступа к «Лань» необходимо зарегистрироваться в системе из локальной сети БНТУ (кнопка «Войти» в правом верхнем углу веб-страницы https://e.lanbook.com/). Для регистрации не из сети БНТУ, Вам необходимо обязательно выбрать организацию в контекстном меню.

Регистрация позволяет пользоваться личным кабинетом, в котором вы можете добавлять книги и статьи в избранное, создавать закладки (добавлять конкретные страницы в избранное), оставлять заметки к записям избранного, создавать и сохранять цитаты из книг и статей.

Работа с текстом и копирование

Возможности работы с текстом:

  • Чтение доступных книг
  • Скачивание приложений к книгам (CD, DVD, архивы)
  • Сохранение и печать фрагментов книг
  • Копирование текста для работы с помощью цитирования

Сохранить или распечатать фрагмент книги можно со страницы просмотра книги. Для этого нажмите значок «Печать» справа на верхней панели страницы просмотра.

В открывшейся форме будет указано, какое количество страниц вы можете использовать. Введите начальную и конечную страницы и нажмите «Сгенерировать файл».

За одну сессию вы не можете использовать страницы сверх установленного для каждой книги предела (для разных книг он может быть разным). Если вам требуется сохранить большее количество страниц, вам следует выйти из Личного кабинета, перезагрузить браузер и зайти снова.

Возможность сохранения или печати фрагмента книги, а также допустимый процент печати устанавливаются каждым правообладателем индивидуально. Это значит, что для некоторых книг данная функция будет недоступна даже при наличии доступа к книге.

Сохранение статей происходит по такому же принципу. Для этого нажмите значок «Печать» в интерфейсе чтения. Вы можете распечатать одну статью за сеанс, для этого нажмите «Сгенерировать файл».

Функция цитирование

Цитирование позволяет выделять и копировать полезную информацию для последующего использования. Скопированные данные можно вставить в текстовый редактор, почту и т. п.

К цитате автоматически будет добавлена библиографическая запись.

Мобильное приложение

Мобильное приложение позволяет скачивать книги и статьи в полном объеме и хранить на устройстве на весь срок действия подписки. Чтение скачанных книг и статей возможно без доступа к интернету.

Рекомендации по дисциплине

Специальный Модуль РПД призван облегчить работу по формированию списков используемой литературы в Рабочих программах дисциплин. Также этот инструмент можно эффективно использовать для оформления литературы в реферативных, исследовательских работах и научных трудах. После заполнения своих дисциплин и указания направления подготовки (по желанию) для вас будет сформирован список, из которого вы можете выбрать нужные книги («Найдено всего/новых»). Этот список обновляется с появлением новых книг, что очень удобно. Из него вы можете выбрать то, что вам необходимо в работе, и в дальнейшем сможете дополнять свой список.

Также вы можете скопировать список рекомендуемых книг по дисциплине и отправить, например, студентам (это публичная ссылка).

Специальный доступ для преподавателей

Специальная программа «Свободный доступ преподавателям» предлагает зарегистрированным пользователям со статусом «Преподаватель» бесплатный персональный доступ ко всем книгам издательства Лань и Планета Музыки, вне зависимости от наличия подписки у вашей организации. Преподаватель может использовать книгу только индивидуально.

Книги определенных издательств можно просматривать внутри тематических разделов в активном режиме “Показывать издательства” над меню каталога книг (верхний левый угол главной страницы поиска).

На постоянной основе на платформе “Лань” осуществляется набор и обучение на онлайн-курсах повышения квалификации для преподавателей вузов и СПО. Обучение является бесплатным и состоит из нескольких модулей. Тематика курсов связана с современными образовательными технологиями.
